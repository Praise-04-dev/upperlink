# 🚀 Project Nebula

A lightweight, blazing-fast utility library for doing things you didn't know you needed — until now.

---

## What is this?

**Project Nebula** is a fictional open-source toolkit built for developers who like clean code, dark mode, and coffee at 2am. It does nothing in particular, but it does it *really well*.

---

## Features

- ⚡ Ridiculously fast (benchmarked against air)
- 🧩 Modular — use only what you need
- 🔒 Secure by default (we think)
- 🌍 Works on every machine except Dave's laptop
- 📦 Zero dependencies (we lied, there are three)

---

## Installation

```bash
npm install project-nebula
```

Or if you're a yarn person:

```bash
yarn add project-nebula
```

---

## Quick Start

```javascript
import { doTheThing } from 'project-nebula';

const result = doTheThing({ magic: true });
console.log(result); // ✨
```

---

## Configuration

| Option      | Type    | Default | Description                          |
|-------------|---------|---------|--------------------------------------|
| `magic`     | boolean | `false` | Enables the magic. Highly recommended.|
| `verbose`   | boolean | `false` | Logs everything. Like, everything.   |
| `darkMode`  | boolean | `true`  | Why would you ever turn this off?    |
| `coffeeMug` | string  | `"full"`| Current mug status. Affects output.  |
